# WIDOKI

Napisz zapytanie w języku SQL do bazy danych HR.

**7.1.** Stwórz widok, który zawiera wyłącznie numer pracownika, imię, nazwisko, nazwę departamentu, adres i miasto. Uporządkuj wyniki alfabetycznie po nazwach miast.

**7.2.** Zmodyfikuj powyższy widok, dodając kolumnę email, w której zamiast pełnych adresów e-mail pracowników znajdzie się literał "*****@****".

**7.3.** Stwórz nowy widok, korzystający z ww. widoku.

**7.4.** Stwórz widok, który zawiera listę pracowników, którzy pracują w działach zawierających wyraz "IT". Uporządkuj wyniki alfabetycznie po nazwiskach pracowników.

**7.5.** Usuń stworzone widoki.
